# 中国理论时间计算器
点击 [这里](.\README_en.md) 阅读英语版本

Click [here](.\README_en.md) to read the English version.

这是一个可以用于计算世界各个地区的理论时间的简易计算器。使用 Python 配合 Streamlit 编写。中国统一采用东八区区时（北京时间）作为全国的法定时间。然而，中国地域辽阔，许多地方与法定时间时差较大，造成计时不便。使用此计算器，可以通过输入北京时间与所在地区的经度，计算当地的理论时间。适合计算中国境内的时间，也可计算世界各地的时间。

本项目已部署在 [jyjcttc.streamlit.app](https://jyjcttc.streamlit.app) 上。
